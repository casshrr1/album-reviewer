# Music Reviewer — REST API

**Base URL (local):** `http://localhost:3000`  
All endpoints return JSON.  
For request bodies, send:

```
Content-Type: application/json
```

---

## Data Models

### Artist
```json
{
  "id": 1,
  "name": "Lil Wayne",
  "genre": "Hip Hop",
  "debut_year": 1999
}
```

### Album
```json
{
  "id": 1,
  "name": "Tha Carter III",
  "genre": "Hip Hop",
  "release_year": 2008,
  "artistId": 1
}
```

### Review
```json
{
  "id": 1,
  "albumId": 6,
  "rating": 4.2,
  "review": "Text...",
  "date": "23.09.2021"
}
```

> `date` is generated by the server in format `DD.MM.YYYY` when creating or updating a review.

---

## Endpoints

## Artists

### List all artists
```
GET /artists
```

**200 OK**
```json
[
  {
    "id": 1,
    "name": "Lil Wayne",
    "genre": "Hip Hop",
    "debut_year": 1999
  }
]
```

---

### Get artist by id
```
GET /artists/:id
```

**200 OK**
```json
{
  "id": 2,
  "name": "A$AP Rocky",
  "genre": "Hip Hop",
  "debut_year": 2011
}
```

**404 Not Found**
```json
{ "error": "Artist not found" }
```

---

### Create artist
```
POST /artists
```

**Body**
```json
{
  "name": "Test Artist",
  "genre": "Hip Hop",
  "debut_year": 2020
}
```

**201 Created**
```json
{
  "id": 10,
  "name": "Test Artist",
  "genre": "Hip Hop",
  "debut_year": 2020
}
```

**400 Bad Request**
```json
{ "error": "Artist already exists" }
```

---

### Delete artist
```
DELETE /artists/:id
```

**200 OK**
```json
{ "message": "Artist deleted successfully" }
```

**404 Not Found**
```json
{ "error": "Artist not found" }
```

## Albums

### List all albums
```
GET /albums
```

**200 OK**
```json
[
  {
    "id": 1,
    "name": "Tha Carter III",
    "genre": "Hip Hop",
    "release_year": 2008,
    "artistId": 1
  }
]
```

---

### Get album by id
```
GET /albums/:id
```

**200 OK**
```json
{
  "id": 3,
  "name": "Wasteland",
  "genre": "R&B",
  "release_year": 2022,
  "artistId": 3
}
```

**404 Not Found**
```json
{ "error": "Album not found" }
```

---

### List albums by artist
```
GET /albums/artist/:artistId
```

**200 OK**
```json
[
  {
    "id": 6,
    "name": "Certified Lover Boy",
    "genre": "Hip Hop",
    "release_year": 2021,
    "artistId": 6
  }
]
```

---

### Create album
```
POST /albums
```

**Body**
```json
{
  "name": "Test Album",
  "genre": "Pop",
  "release_year": 2024,
  "artistId": 1
}
```

**201 Created**
```json
{
  "id": 15,
  "name": "Test Album",
  "genre": "Pop",
  "release_year": 2024,
  "artistId": 1
}
```

**400 Bad Request**
```json
{ "error": "Album must have a name and artistId" }
```
or
```json
{ "error": "Release year can't be in the future" }
```

---

### Delete album
```
DELETE /albums/:id
```

**200 OK**
```json
{ "message": "Album deleted successfully" }
```

**404 Not Found**
```json
{ "error": "Album not found" }
```

---

## Reviews

### List reviews for an album
```
GET /reviews/album/:albumId
```

**200 OK**
```json
[
  {
    "id": 5,
    "albumId": 1,
    "rating": 8.5,
    "review": "this album is really growing on me!",
    "date": "26.01.2026"
  }
]
```

---

### Create review
```
POST /reviews
```

**Body**
```json
{
  "albumId": 1,
  "rating": 9,
  "review": "Great album"
}
```

**201 Created**
```json
{
  "id": 25,
  "albumId": 1,
  "rating": 9,
  "review": "Great album",
  "date": "29.01.2026"
}
```

**400 Bad Request**
```json
{ "error": "Album not found" }
```
or
```json
{ "error": "Rating has to be between 0 and 10" }
```

---

### Update review
```
PUT /reviews/:id
```

**Body**
```json
{
  "rating": 8.7,
  "review": "Updated text"
}
```

**200 OK**
```json
{
  "id": 2,
  "albumId": 6,
  "rating": 8.7,
  "review": "Updated text",
  "date": "29.01.2026"
}
```

---

### Delete review
```
DELETE /reviews/:id
```

**200 OK**
```json
{ "message": "Review deleted successfully" }
```

**404 Not Found**
```json
{ "error": "Review not found" }
```

## Frontend Usage

The client uses `fetch` through a `safeFetch()` helper:

- **Artists**
  - `GET  /artists`
  - `POST /artists`
  - `DELETE /artists/:id`

- **Albums**
  - `GET /albums/artist/:artistId`
  - `POST /albums`
  - `DELETE /albums/:id`

- **Reviews**
  - `GET /reviews/album/:albumId`
  - `POST /reviews`
  - `PUT /reviews/:id`
  - `DELETE /reviews/:id`

---

## Status Codes

- `200 OK` - successful GET / update / delete  
- `201 Created` — successful create  
- `400 Bad Request` — validation errors  
- `404 Not Found` — entity does not exist  
- `500 Internal Server Error` — unexpected server errors
